Quick start

import sample

Installation
------------
The easiest way to install sample is with pip
```
pip install sample
```

Alternately, you can clone the code repository and install from source
```
git clone git@gitlab.info-ufr.univ-montp2.fr:daniele.coslovich/sample.git
cd sample
make install
```
